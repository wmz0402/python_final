# 🔐 仓库管理系统
## ✨ 核心功能特性

- **👥 用户与权限管理 (RBAC)**
  - 支持 `普通员工` 和 `系统管理员` 两种身份。
  - 密码加密存储（SHA-256），支持通过设置密保问题重置密码。
  - 敏感操作（如下架商品、导出报表、查看日志）仅限管理员可用。
- **📦 商品进销存管理**
  - 支持 **普通商品** 与 **易腐商品**（附带保质期属性）。
  - 标准化商品 ID 格式校验（如：`P001`）。
  - 完善的入库（新增/补货）、出库逻辑，以及库存不足等异常拦截。
- **🔍 智能查询与排序**
  - 支持对商品名称或 ID 进行模糊搜索 。
  - 支持按商品单价或库存数量进行升序/降序排列 。
- **⚠️ 系统智能预警**
  - **库存预警**：当商品库存不足 5 件时触发。
  - **临期预警**：当易腐商品保质期 ≤ 7 天时触发。
- **📊 可视化与报表**
  - 终端表格化展示（利用 `PrettyTable`）。
  - 支持生成库存分布的柱状图（利用 `Matplotlib`）。
  - 支持将库存数据一键导出为 `.csv` 文件，方便 Excel 查阅（防止中文乱码处理）。
- **💾 数据持久化与日志跟踪**
  - 所有数据自动存入本地的 `data/` 目录（自动创建）。
  - 使用 `JSON` 格式存储库存(`inventory.json`)、用户(`users.json`)和操作日志(`log.json`)。

## 🚀 运行指南

 **运行程序**
   ```bash
   python main.py
   ```
